# SandwichTools

## Backend - API Server
### Installation

```bash
$ npm install
```

### Settings
#### .env

```bash
# set API Server Port like this.
PORT=4000
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Frontend
### Installation

```bash
$ npm install
```

### Settings
#### .env

```bash
# set api server domain url like this.
REACT_APP_DOMAIN_URL=http://localhost:4000
```

### Running the app

```bash
# development
$ npm run start:dev

# production mode
$ npm run start

# build
$ npm run build
```

### Running on Browser
```bash
localhost:3000
```
